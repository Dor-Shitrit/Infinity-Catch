## About the Game

**Infinity Catch** is a local multiplayer arcade game built in **Processing**.

Two players compete on the same screen:

- **Player 1, Thanos** uses the keyboard
- **Player 2, Iron Man** uses the mouse

Falling objects appear from the top of the screen, and both players race to catch them before they hit the ground.

### Objective

- Catch stones to gain points
- Avoid missing stones, because both players lose a life when a stone reaches the bottom
- Be the first to reach **10 points**

---

## Gameplay Rules

- Each player starts with **3 lives**
- **3 falling objects** are active at the same time
- Catching a falling object gives the player **1 point**
- Missing a falling object removes **1 life from both players**
- If **Thanos** reaches **10 points**, Player 1 wins
- If **Iron Man** reaches **10 points**, Player 2 wins
- If both players lose all lives, the game ends with **Game Over**

---

## Controls

### Player 1, Thanos
| Key | Action |
|---|---|
| Left Arrow | Move left |
| Right Arrow | Move right |
| Up Arrow | Stop |

### Player 2, Iron Man
| Input | Action |
|---|---|
| Mouse movement | Move horizontally |

### Start the Game
- Press **any key**
- Or click the **mouse**
